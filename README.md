# tbrelo.com — Tampa Bay Relocation Guide

## Quick Start (Do This First)

```bash
# 1. Clone the repo
git clone https://github.com/bamahe/tbrelo.git
cd tbrelo

# 2. Install dependencies
npm install

# 3. Run locally
npm run dev

# 4. Open http://localhost:3000
```

## How to Add Content

Drop a markdown `.md` file in the right folder. That's it.

| Content Type | Folder | URL Pattern |
|---|---|---|
| County hubs | `content/counties/` | `/counties/hillsborough/` |
| City pages | `content/cities/` | `/cities/brandon/` |
| Blog posts | `content/blog/` | `/blog/my-post-slug/` |
| Pillar pages | `content/pillar/` | `/moving-to-tampa-bay/` |
| Moving-from pages | `content/moving-from/` | `/moving-from/new-york/` |
| Static pages | `content/pages/` | `/affiliate-disclosure/` |

### Markdown Template

Every `.md` file needs this frontmatter at the top:

```markdown
---
title: "Moving to Brandon, Florida"
metaTitle: "Moving to Brandon FL — Relocation Guide (2026)"
metaDescription: "Everything about relocating to Brandon, FL. Cost of living, schools, neighborhoods, and honest local insight."
type: city
county: Hillsborough
publishedAt: "2026-03-12"
updatedAt: "2026-03-12"
---

Your content here. Use {{adt}}, {{pods}}, {{nowtb}}, etc. for affiliate links.
They auto-convert to tracked links.
```

## Affiliate Links

All affiliate links live in `lib/affiliates.ts`. Change a URL there → it updates everywhere.

Use `{{key}}` in any markdown file:
- `{{nowtb}}` → Barrett Henry REALTOR link
- `{{bbs}}` → Best Bay Services link
- `{{adt}}` → ADT affiliate link
- `{{pods}}` → PODS affiliate link
- See `lib/affiliates.ts` for the full list

## Deploy to Vercel

```bash
# First time: connect to Vercel
npx vercel

# After that, every git push auto-deploys
git add .
git commit -m "add new content"
git push
```

## DNS Setup (Namecheap → Vercel)

1. Go to Namecheap → Domain List → tbrelo.com → Advanced DNS
2. Delete all existing records
3. Add: A Record → @ → 76.76.21.21
4. Add: CNAME → www → cname.vercel-dns.com
5. In Vercel dashboard → Settings → Domains → Add tbrelo.com
